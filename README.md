# trash-detection-yolov8

YOLOv8 tabanlı **trash object detection** projesi.  
Model eğitimi, TensorFlow Lite export ve **Android (Kotlin) on-device inference** içerir.

## Model

GitHub dosya boyutu limitleri nedeniyle **`model.tflite`** repo içinde yer almaz.

Mobilde kullanmak için:
1. **Releases** bölümünden `model.tflite` dosyasını indirin.
2. Dosyayı aşağıdaki dizine kopyalayın:
android_app/app/src/main/assets/


Dosya adı **tam olarak** `model.tflite` olmalıdır.
